# M5Cardputer_WebRadio
Exemplo do Webradio para Esp8266 convertido para M5 Cardputer 

comandos:
BtnGO click rapido:  muda de estação
BtnGO Hold: Aumenta o volume
BtnGO dois clicks rapidos e Hold: diminue o volume
